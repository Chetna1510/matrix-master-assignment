Mock Algorithm Challenge

You will complete 5 algorithm problems using pseudocode, Python code, and T-diagrams where required.

Algorithms

1. This README corresponds to the following algorithms:

    Shift array elements to the front

    Example: [2,1,6,4,-7] → [-7,4,6,1,2]

2. Remove negative numbers recursively

Example: [1,-2,4,1] → [1,4,1]

3. Find the longest word in a string

4. Return an array of largest numbers from sub-arrays

    Example input:
    [[13,27,18,26], [32,35,37,39], [1000,1001,857,1]]

    Output: [27, 39, 1001]

5. Title-case a string (capitalize first letter of each word)
