
# React Todo List / Task Manager

A simple React application to manage tasks with priorities. Users can add, delete, and view tasks in a table format. Each task has a priority (Low, Medium, High) and a status.

## Features

- Add a new task with a title and priority
- Delete tasks
- View all tasks in a table
- Task status display (Pending by default)
- Built using **React Class Components** and **state management**
